항상 눈에보이는 결과물을 만들어내는것에 흥미를 느끼는 신입 개발자 박태현입니다.
이전에 퍼블리셔로 근무하며 여러가지 기능들을 구현하면서 새로운 것들을 학습 할 수 있다는것, 문제가 생겨도 해결하는 과정에서 많을 것을 배울 수 있다는 것에서 성취감과 계속해서 성장할 수 있음에 매력을 느껴 프론트엔드 개발자를 꿈꾸게 되었습니다.
끈기있는 태도와 꾸준함으로 앞으로도 성장하는, 성장가능성이 무궁무진한 신입 개발자가 되도록하겠습니다.

STACK

FRONT-END

  • HTML5

    디자인 시안에 맞게 효율적인 구조를 작성할 수 있으며 웹표준성, 웹접근성을 준수합니다.

  • CSS3

    레이아웃 디자인을 위한 기본적인 속성들을 적절히 사용할 수 있으며 반응형작업을 할 수 있습니다.

  • SASS

    SCSS 사용법과 Mixin, Variables를 사용하여 효율적인 스타일 작업을 할 수 있습니다.

  • TAILWIND CSS

    tailwind css 문법을 활용하여 className으로 스타일작업이 가능합니다.

  • JAVASCRIPT

    DOM, 이벤트 처리방식을 이해하며 api 호출을 위한 비동기 작업 등의 코드를 작성 할 수 있습니다.

  • JQUERY

    클릭, 스크롤 등 기본적이 이벤트 제어가 가능하고 swiper 등 라이브러리를 사용할 수 있습니다.

  • TYPESCRIPT

    기본적인 타입을 생성해 리액트 작업에서 적용시켜 작업 할 수 있습니다.

  • REACT

    SPA 개념을 이해하고 있으며, 재사용이 가능한 컴포넌트들을 만들수 있고 제공하는 Hooks들을 활용할 수 있습니다.

  • NEXT.JS

    SSR의 개념을 이해하고 있으며, NEXTJS의 규칙을 따라 서버컴포넌트와 클라이언트컴포넌트를 만들어 개발할 수 있습니다.

  • REDUX

    Store를 생성하고, Reducer 함수를 정의해 전역 상태관리를 할 수 있습니다.

  • RECOIL

    Store를 생성하고 Recoil hooks를 활용하여 상태관리를 할 수 있습니다.
